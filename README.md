# IRONFLOW 1.0 Production

โปรเจกต์ใหม่สะอาดสำหรับ Vercel

## ก่อนใช้งาน

ห้ามอัปโหลดโฟลเดอร์เหล่านี้ขึ้น GitHub:

- node_modules
- .next
- .vercel

ไฟล์ `.gitignore` ใส่ไว้ให้แล้ว

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

หลังจาก build ผ่าน:

```bash
git add .
git commit -m "IRONFLOW 1.0 production"
git push
```
